Zepto / Blinkit Grocery Analysis — Power BI Dashboard

 Overview
An interactive Power BI dashboard analyzing grocery sales data 
across outlet types, locations, and item categories.

 Key Metrics
- Total Sales (Gauge)
- Avg Sales per Outlet
- Item Visibility
- Customer Ratings

 Visuals Used
- Clustered Bar Chart: Sales by Item Type
- Donut Chart: Sales by Outlet Location
- Funnel Chart: Sales by Outlet Size
- Area Chart: Ratings by Item Type
- Slicers: Item Type, Outlet Size, Location, Year

 Tools
- Power BI Desktop
- Dataset: BlinkIT Grocery Data (Kaggle)

 How to open
Download the `.pbix` file and open with Power BI Desktop (free).
